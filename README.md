# TOYWeb

React website for Transformative Opportunities for Youth (TOY), configured for GitHub Pages.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Create a GitHub repository named `TOYWeb`.
2. Update `homepage` in `package.json` with your GitHub username.
3. Push this project to GitHub.
4. Run:

```bash
npm run deploy
```

5. In GitHub repo settings, make sure GitHub Pages is configured to serve from the `gh-pages` branch if needed.
